1- Responder las siguientes preguntas:
a- ¿Qué es JSX?
Jsx es una extension de js que nos permite escribir codigo similar a 
html pero tambien usando sintaxis de js. Es como una mezcla de los dos lenguajes


b- ¿Qué es un estado en React?
El estado en react es la instancia actual de algun componente en un momento especifico
Este puede ser modificado con una funcion dedicada para ese trabajo


c- ¿Qué es un componente?
Un componente en React es una porcion de codigo que podemos reutilizar inyectandolo en 
nuestro html


d- ¿Para qué sirve el hook useState?
el hook useState recibe dos parametros, el primer parametro sirve para setear un estado
de un componente y el segundo parametro puede cambiar el estado inicial


e- ¿Para qué sirve el hook useEffect?
El hook useEffect sirve para definir un efecto que ejecuta una porcion de codigo en un 
momento especifico cuando reciba un cambio 


f- ¿Qué es el contexto en React?
El contexto es un elemento que podemos crear en React para establecer una comunicación
directa entre un componente en un nivel muy alto y uno en un nivel mucho más bajo.
